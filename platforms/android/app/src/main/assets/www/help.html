About Class Track

Class Track is a app for students and teachers. They can memorized class routine, exam schedule, assignment schedule for each subject. Also add meetings. Subject may be guitar class, cricket class etc. not mandatory subject must be our education related.

Subject

You need to add subject which subject you want to memorize for classes, exam, assignment. If other module have any data except meeting module then please don't update subject because previous data will show in other module and the subject new information could not be access by other module in this Class Track version. Update that subject which don't have any classes, exam and assignment. This is not a bug but this is a problem. Next Version Developer will fix this problem :D :P. But you can also update other module information by a little tricks. At first udate your subject which you want to change and go other module where the subject have. Each data for that subject click Edit button and change any other subject. And again press edit button and update previous subject which change one step ago.

Class

Basically Student need to fill out all the fields except Note. But you can use as your wish. Note field mainly for teachers. Teacher can save which batch he/she will be teach.

Assignment

You can add assignment which you want to memorize.

Exam

You can add exam which you want to memorize.

Meeting

You can add meeting which you want to memorize.
